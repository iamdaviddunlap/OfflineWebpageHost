Introduction<\/h2>\n\n\n

This one\u2019s a little late. I had a multi-day power outage when the playtest doc was released, and I\u2019m just now getting my thoughts written down.<\/p>\n\n\n\n

This round\u2019s playtest document includes new versions of druid and paladin. These are the remaining two classes in the \u201cpriest\u201d group after we saw the cleric in the previous round, and both classes got a lot of changes. Some are simplifications, some aim to address problem cases in the classes, and some are\u2026 more conceptually explorative.<\/p>\n\n\n

Reading Tea Leaves <\/h2>\n\n\n

\u201cYou might find an older subclass doesn\u2019t fully work with the features in the playtest version of a class. If we publish the new version of the class, we\u2019ll resolve that discrepancy.\u201d<\/p>\n\n\n\n

OneD&D is moving toward all classes getting subclass features at the same level. This means that everyone\u2019s capabilities \u201cspike\u201d at those same levels, allowing the design of the game to accommodate that progression rather than being a flat progression at every level except 5 among other likely changes to the game.<\/p>\n\n\n\n

This creates the discrepancy between 5e subclasses and OneD&D subclasses. There are two ways to \u201cresolve\u201d the discrepancy: revert to the subclass feature progression of the 2014 classes, or provide updated versions of the existing subclasses.<\/p>\n\n\n\n

Does this mean a conversion guide for existing subclasses? Or just that we\u2019ll see reprints in future supplements? Unclear. A conversion guide would mean that everyone\u2019s 5e books don\u2019t become completely obsolete when OneD&D releases, and the existing 5e books might continue to sell well into OneD&D\u2019s run. Reprints would mean that WotC gets to sell us a new book or two, but the community will justifiably complain about them doing so.<\/p>\n\n\n
