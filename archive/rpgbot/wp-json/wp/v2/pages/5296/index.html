Introduction<\/h2>\n\n\n

The Oracle’s choice of Mystery defines the character as much as any archetype, providing new skills, spells, and a list of available revelations. Because there is no overlap in revelations between Mysteries, each Mystery makes the Oracle very unique. Note that Oracles get 6 Revelations normally, but can pick up more with the Extra Revelation feat. Also note that you cannot select a revelation more than once according to The FAQ<\/a>.<\/p>\n\n\n
